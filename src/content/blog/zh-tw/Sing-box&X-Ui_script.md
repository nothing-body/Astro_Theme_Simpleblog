此腳本是簡單功能較少但一眼就能看懂的部屬腳本：

- [Github 作者233boy](https://github.com/233boy/sing-box)
- [安装及使用：](https://233boy.com/sing-box/sing-box-script/)
- **_ 教學文內沒提到添加socks5方式 添加方式是`sb add socks` _**

此腳本是甬哥製作的精裝腳本，內含許多功能的整合：

- [Github 作者yonggekk](https://github.com/yonggekk/x-ui-single-server)
- [甬哥部落格教學文](https://www.ioiox.com/archives/653.html)
- [Youtube影片](https://www.youtube.com/watch?v=aGEmCu503V8)

甬哥的另一個功能少但更簡便的腳本：

- [Argosbx](https://github.com/yonggekkk/argosbx)

腳本支持的協議：AnyTLS、Any-reality、Vless-xhttp-reality-vison-enc、Vless-tcp-reality-vision、Vless-xhttp-vison-enc、Vless-ws-vision-enc、Shadowsocks-2022、Vmess-ws、Socks5、Hysteria2、Tuic、Argo臨時/固定隧道支持Vless-ws-vision-enc或Vmess-ws

此腳本能使用網頁生成SSH一鍵腳本指令：[SSH一鍵腳本生成頁面](https://yonggekkk.github.io/argosbx/)
