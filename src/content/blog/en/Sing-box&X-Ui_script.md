This is a simple deployment script with fewer features but is very easy to understand at a glance:

- [Github Author: 233boy](https://github.com/233boy/sing-box)
- [Installation and Usage:](https://233boy.com/sing-box/sing-box-script/)
- - The tutorial doesn't mention how to add socks5. The way to add it is `sb add socks`

This script is a comprehensive script created by Yongge, integrating many features:

- [Github Author: yonggekk](https://github.com/yonggekk/x-ui-single-server)
- [Yongge's Blog Tutorial](https://www.ioiox.com/archives/653.html)
- [Youtube](https://www.youtube.com/watch?v=aGEmCu503V8)

Another script by Yongge that has fewer features but is even simpler:

- [Argosbx](https://github.com/yonggekkk/argosbx)

Protocols supported by the script: AnyTLS, Any-reality, Vless-xhttp-reality-vison-enc, Vless-tcp-reality-vision, Vless-xhttp-vison-enc, Vless-ws-vision-enc, Shadowsocks-2022, Vmess-ws, Socks5, Hysteria2, Tuic, Argo temporary/fixed tunnel supports Vless-ws-vision-enc or Vmess-ws

This script can generate SSH one-click commands using a web page: [SSH One-Click Script Generation Page](https://yonggekkk.github.io/argosbx/)
