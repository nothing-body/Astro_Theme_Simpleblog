## 实际用户反馈重点

具体用户留言比平均分数更能补足细节。较正面的 G2 类反馈多集中在设置快速、客服有帮助，以及可以从同一个 dashboard 购买不同代理类型或指定地区。这对已经知道自己需要哪个国家、协议与代理类别的买家，比单纯看星级更有参考价值。

负面留言则指出实务风险。Trustpilot 近期可见案例包括：用户报告 IP 被意外替换、替换后国家不一致、客服处理速度不如预期，以及 residential proxies 在目标服务或第三方 IP database 里被分类成不同类型。这些留言重要，是因为代理是否成功常取决于特定目标网站，而不只是 proxy 能不能连接。

实际购买时，建议按用途阅读留言。需要特定国家 static IP 的买家，应特别注意替换地区与 subnet 稳定性；使用 residential traffic 的买家，则应在扩大购买前先测目标平台是否把 IP 识别成预期类型。

## 退款与条款注意

Proxy-Seller 的 [Return Policy](https://proxy-seller.com/zh/return/) 说明服务采用预付款。政策列出在特定情况下可于 72 小时内申请扣除费用后的全额退款，例如代理不可用且无法替换、无法连接且经技术人员确认、缺少所需 IP 地区、缺少所选用途需要的 IP，或购买用途本身被条款禁止。政策也说明周方案通常超过 24 小时后不能退款，3 天试用 residential proxies 通常不能退款。

[Public Offer](https://proxy-seller.com/zh/offer/) 说明，使用服务代表接受协议与退款政策；其中也禁止某些用途，例如未经许可为 benchmark 或竞争监测而访问服务。

## 适合谁

Proxy-Seller 比较适合需要大量国家/地区静态代理、想在 IPv4/IPv6/ISP/residential/mobile 之间选择，并重视手动地区选择、dashboard 与 API 工作流的人。它的国家页很多，明显偏向已经知道自己要哪个地区与哪种代理类型的买家。

如果你需要非常透明的固定价格、不想进 selector 才知道最终价、需要某个第三方平台保证识别为特定 IP 类型，或不能接受 IP 替换与地区变动，就要先小额测试。放大购买前，务必用自己的目标服务验证 IP。

## 购买前检查

1. 确认国家、数量、期限与代理类型的最终结账价格。
2. 询问客服你的目标平台是否适合该代理类型。
3. 确认方案是试用、周付、月付还是长期，因为退款规则不同。
4. 用多个 database 与实际目标服务检查 IP classification。
5. 付款前保留地区需求与客服承诺记录。
6. 不要只依赖 Trustpilot，因为该页有 guideline-breach warning。
