## OpenClaw 是什么？

OpenClaw 是一款整合了多种 AI 工具的软件平台，其核心设计结合了 Skill Agent（技能代理）、工具链（Tool Chain）等模块，让用户可以通过统一的界面调用不同的 AI 功能，例如代码生成、网页浏览、文件操作、命令执行等。

对我来说，OpenClaw 其实就是 Skill Agent 等工具集合起来的整合软件，除了 OpenClaw，市面上还有更多类似的软件。

## 一键整合包的潜在风险

走红后坊间流传着大量“一键整合包”，部分整合包已被社区举报内含恶意程序，常见的风险类型包括：

- **远程访问木马（RAT）**：安装后攻击者可在后台远程控制你的电脑。
- **后门程序（Backdoor）**：在系统中留下隐藏入口，供日后随时入侵。
- **挖矿程序（Cryptominer）**：偷偷占用你的 CPU / GPU 资源替他人挖加密货币。
- **凭据窃取（Credential Stealer）**：截取浏览器内存储的账号密码、Cookie 等敏感信息。

建议只从 **官方 GitHub 仓库** 或官方文档的安装指引进行安装，避免使用来路不明的整合包脚本。
