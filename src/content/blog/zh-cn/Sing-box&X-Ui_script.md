此脚本是简单、功能较少但一眼就能看懂的部署脚本：

- [Github 作者 233boy](https://github.com/233boy/sing-box)
- [安装及使用说明](https://233boy.com/sing-box/sing-box-script/)
- **_ 教程文章内没提到添加 socks5 的方式，添加方式为：`sb add socks` _**

此脚本是甬哥制作的精装脚本，内含许多功能的整合：

- [Github 作者 yonggekk](https://github.com/yonggekk/x-ui-single-server)
- [甬哥博客教程文章](https://www.ioiox.com/archives/653.html)
- [Youtube 视频](https://www.youtube.com/watch?v=aGEmCu503V8)

甬哥的另一个功能少但更简便的脚本：

- [Argosbx](https://github.com/yonggekkk/argosbx)

脚本支持的协议：AnyTLS、Any-reality、Vless-xhttp-reality-vison-enc、Vless-tcp-reality-vision、Vless-xhttp-vison-enc、Vless-ws-vision-enc、Shadowsocks-2022、Vmess-ws、Socks5、Hysteria2、Tuic。Argo 临时/固定隧道支持 Vless-ws-vision-enc 或 Vmess-ws。

此脚本可以使用网页生成 SSH 一键脚本命令：[SSH 一键脚本生成页面](https://yonggekkk.github.io/argosbx/)
